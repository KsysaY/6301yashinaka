# Лабораторная работа №5

## Основные возможности:
Prepare: Фильтрация экспонатов из CSV-файла и подготовка JSON-списка для скачивания.
Process: Асинхронная загрузка изображений и их параллельная обработка (свёртка, размытие, фильтр Собеля) на ядрах CPU.
Analyze: Статистический анализ датасета и построение графиков распределения объектов.

## Установка
### 1. Установка из TestPyPI
Пакет доступен в тестовом репозитории Python. Для корректной установки зависимостей используйте команду:
#### pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ 6301yashinaka

### 2. Локальная установка для разработки
Если вы скачали исходный код, установите его в режиме редактирования:
#### pip install -e .

### Использование (CLI)
Программа управляется через единую команду metetl. Для вывода справки введите:
#### metetl --help

## Шаг 1: Подготовка данных
Отбор картин из исходного CSV файла музея:
#### metetl prepare --csv data/MetObjects.csv --output data/to_download.json --num 20

## Шаг 2: Скачивание и обработка
Запуск асинхронного пайплайна загрузки и параллельной фильтрации:
#### metetl process --input data/to_download.json --output images --num 5

## Шаг 3: Анализ датасета
Генерация статистических графиков:
#### metetl analyze --csv data/MetObjects.csv --output-dir data/plots
